# Bank Marketing Campaign Analysis

## Project Overview
This project analyzes a bank marketing dataset to identify factors that influence customer subscription to term deposits.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Project Tasks
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Visualization
- Customer Behavior Analysis

## Dataset
Bank Marketing Dataset
